# Contributing to Event Registration & Ticket QR Scanner

## Setup Instructions for Contributors

1. Clone the repository
2. Follow the setup instructions in the main README.md
3. Make sure all tests pass before submitting a PR
4. Follow the coding standards (TypeScript, NestJS, React conventions)

## Project Structure

- `backend/` - NestJS backend API
- `frontend/admin-app/` - Admin React application
- `frontend/organizer-app/` - Organizer React application  
- `frontend/attendee-app/` - Attendee React application

## Git Workflow

1. Create a feature branch from `main`
2. Make your changes
3. Test thoroughly
4. Submit a pull request

## Code Style

- Use TypeScript strict mode
- Follow NestJS and React best practices
- Use meaningful variable and function names
- Add comments for complex logic
- Update documentation as needed
