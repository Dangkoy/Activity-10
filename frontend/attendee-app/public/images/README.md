# Event Background Image

Place your `event_hd.jpg` image file here.

**Supported formats:**
- JPG/JPEG (recommended)
- PNG
- WebP

**Recommended size:**
- Minimum: 1920x1080px (Full HD)
- Optimal: 2560x1440px (2K) or 3840x2160px (4K)

**File name:** `event_hd.jpg`

The image will be used as the background wallpaper for the Events page (starting page) in the Attendee app.

**Note:** The image will have a semi-transparent overlay for better text readability.
