# Login Background Image

Place your background image here and name it `login-background.jpg`

**Supported formats:**
- JPG/JPEG
- PNG
- WebP

**Recommended size:**
- Minimum: 1920x1080px
- Optimal: 2560x1440px or higher

**File name:** `login-background.jpg`

The image will be used as the background for the organizer login page.
